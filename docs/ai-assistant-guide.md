# AI 智能助手使用指南

## 功能概述

AI 智能助手是由 **DeepSeek 大模型** 驱动的专属VIP功能，为用户提供智能对话、代码生成、内容创作等强大的AI能力。

## 🌟 功能特性

### 核心能力

1. **智能对话** 💬
   - 自然语言理解
   - 上下文记忆
   - 多轮对话支持

2. **代码生成** 💻
   - 支持多种编程语言
   - 代码解释与优化
   - Bug 修复建议

3. **内容创作** ✍️
   - 文章撰写
   - 文案创意
   - 翻译润色

4. **数据分析** 📊
   - 数据解读
   - 趋势分析
   - 建议方案

5. **知识问答** 📚
   - 专业知识解答
   - 学习辅导
   - 概念解释

## 🔐 访问权限

### 权限要求

AI 助手功能仅对以下用户开放：
- ✅ **VIP 年卡**用户
- ✅ **SVIP 年卡**用户

### 为什么需要年卡？

- AI 服务需要消耗大量算力资源
- 年卡用户享有更高的 Token 配额
- 保证服务质量和响应速度
- 支持产品持续发展

### 如何获取权限？

1. 点击右上角用户菜单
2. 选择"VIP 助手"（如果没有显示说明权限不足）
3. 如果没有权限，会看到升级提示
4. 点击"立即升级 VIP"购买年卡套餐

## 📖 使用教程

### 第一步：访问 AI 助手

**方式一：从首页进入**
1. 登录系统
2. 点击右上角用户头像
3. 在下拉菜单中选择"AI 助手"（带VIP标签）

**方式二：直接访问**
- URL: `/ai-assistant`

### 第二步：开始对话

#### 快速开始

首次进入会看到欢迎屏幕，提供4个快捷操作：

1. **编写代码** - 帮您生成代码示例
2. **知识问答** - 回答专业问题
3. **数据分析** - 分析数据趋势
4. **内容创作** - 撰写文章内容

点击任意卡片即可快速开始对话。

#### 自由输入

在底部输入框中输入您的问题：
- 按 **Enter** 发送消息
- 按 **Shift + Enter** 换行
- 支持长文本输入

### 第三步：查看回复

AI 会在几秒内回复您的问题，回复内容支持：
- ✅ Markdown 格式
- ✅ 代码高亮
- ✅ 数学公式
- ✅ 列表和表格

### 第四步：互动操作

对于 AI 的每条回复，您可以：
- **复制** - 复制回复内容到剪贴板
- **点赞** - 标记有用的回复

## 🎯 使用技巧

### 1. 提问技巧

**好的提问示例：**
```
❌ 差：写个爬虫
✅ 好：用Python写一个爬取豆瓣电影Top250的爬虫，包含异常处理和数据保存

❌ 差：这个代码有问题
✅ 好：这段Python代码报错"IndexError: list index out of range"，帮我找出原因并修复

❌ 差：写篇文章
✅ 好：写一篇800字的文章，主题是"人工智能在医疗领域的应用"，包括现状、挑战和未来展望
```

**提问原则：**
- 🎯 **明确具体** - 清晰描述您的需求
- 📋 **提供上下文** - 说明相关背景信息
- 💡 **一次一个** - 每次只问一个问题
- 🔄 **逐步细化** - 根据回复逐步深入

### 2. 代码相关

**请求代码时：**
- 指定编程语言
- 说明功能需求
- 提供输入输出示例
- 说明是否需要注释

**示例：**
```
用JavaScript写一个防抖函数，要求：
1. 支持设置延迟时间
2. 支持立即执行选项
3. 包含详细注释
```

### 3. 调试代码

**正确方式：**
```
这段代码有bug：

[粘贴您的代码]

错误信息：[错误内容]
预期行为：[期望的结果]
实际行为：[实际的结果]
```

### 4. 多轮对话

AI 会记住上下文，您可以：
- 追问细节
- 要求修改
- 深入探讨

**示例对话：**
```
用户：写一个计算器类
AI：[提供代码]
用户：加上历史记录功能
AI：[修改代码]
用户：再加上撤销功能
AI：[再次修改]
```

## 💾 对话管理

- 自动保存每次对话
- 点击可切换查看

### 新建对话

点击左上角 ➕ 按钮创建新对话：
- 开始全新话题
- 独立的上下文

每个对话旁有删除按钮：
- 点击删除按钮
- 确认后永久删除
- 无法恢复，请谨慎操作

### 清空当前对话

底部输入框左侧有清空按钮：
- 清空当前对话所有消息
- 保留对话记录
- 可以重新开始

## 📊 Token 管理

### 什么是 Token？

Token 是 AI 处理文本的基本单位：
- 1 个 Token ≈ 0.75 个英文单词
- 1 个中文字 ≈ 1-2 个 Token

### Token 配额

**VIP 年卡用户：**
- 每月 **100,000 Tokens**
- 约等于 **15万汉字**
- 每月自动重置

**SVIP 年卡用户：**
- 每月 **200,000 Tokens**
- 约等于 **30万汉字**
- 优先级更高

### 查看使用情况

底部提示栏显示：
```
Token 使用情况: 1,234 / 100,000
```

### 节省 Token 技巧

1. **精简提问** - 避免冗余描述
2. **分段对话** - 复杂问题分多次问
3. **新建对话** - 不相关话题开新对话
4. **及时清空** - 不需要的对话及时删除

## 🔧 常见问题

### Q1: 为什么我看不到 AI 助手？

**A:** 需要满足以下条件：
1. 已登录账户
2. 拥有 VIP 或 SVIP 会员
3. 会员类型是**年卡**（不是月卡或季卡）

### Q2: AI 响应很慢？

**A:** 可能的原因：
- 网络连接不稳定
- 服务器负载较高
- Token 配额不足
- 建议稍后重试

### Q3: AI 回答不准确？

**A:** 解决方法：
- 更清晰地描述问题
- 提供更多上下文
- 分步骤提问
- 验证关键信息

### Q4: Token 用完了怎么办？

**A:** 
- 等待下月自动重置
- 升级到 SVIP 获得更多配额
- 优化提问方式节省使用

### Q5: 对话历史会保存多久？

**A:** 
- 本地浏览器永久保存
- 清除浏览器数据会丢失
- 建议重要内容及时复制保存

### Q6: 可以用 AI 做什么？

**A:** 适合场景：
- ✅ 编程学习和开发
- ✅ 文案创作和润色
- ✅ 数据分析和解读
- ✅ 知识学习和答疑
- ✅ 创意灵感和建议

**不适合场景：**
- ❌ 医疗诊断建议
- ❌ 法律专业意见
- ❌ 投资理财决策
- ❌ 紧急事件处理

### Q7: 如何给 AI 反馈？

**A:** 
- 点击消息下方的"赞"按钮
- 联系客服反馈使用体验
- 提出功能改进建议
