## 📖 功能详解

### 🌐 联网搜索功能

#### 什么是联网搜索？
联网搜索功能让 AI 能够在回答问题前，先搜索互联网获取最新信息，然后基于搜索结果和自身知识给出综合性的回答。

#### 适用场景

✅ **最适合：**
- 查询最新新闻和时事
- 获取实时数据和统计
- 了解最新技术趋势
- 查找当前的价格、天气等动态信息
- 验证最新的事实和数据

❌ **不适合：**
- 纯理论性问题
- 代码编写（不需要最新信息）
- 数学计算
- 历史事件（已知信息）

#### 使用方法

1. **开启功能**
   - 在输入框上方找到 "AI 模式" 工具栏
   - 点击 "联网搜索" 按钮（带地球图标）
   - 按钮变为蓝色表示已启用
   - 右侧会显示 "实时搜索" 状态

2. **输入问题**
   ```
   ✅ 好例子：
   - "2024年人工智能领域有哪些重大突破？"
   - "最新的iPhone价格是多少？"
   - "今天北京的天气怎么样？"
   - "比特币当前价格"
   ```

3. **获取回答**
   - AI 会先搜索互联网（显示搜索状态）
   - 然后基于搜索结果给出回答
   - 回答中会引用搜索来源

4. **查看标记**
   - 您的消息旁会显示 "联网" 蓝色标签
   - 表明这条消息使用了联网搜索功能

#### 功能特点

- ⚡ **实时搜索**：获取最新互联网信息
- 🎯 **智能筛选**：AI 自动筛选相关结果
- 📚 **来源引用**：回答中会引用信息来源
- 🔄 **动态更新**：每次搜索都获取最新数据

#### 注意事项

⚠️ **使用提示：**
- 联网搜索会增加响应时间（约 1-2 秒）
- 搜索结果质量取决于互联网可用信息
- 某些地区或网络可能影响搜索效果
- 功能标记为 BETA，持续优化中

### 🧠 深度思考功能

#### 什么是深度思考？
深度思考模式让 AI 进行更深层次的推理和分析，提供更详细、更全面、更有深度的回答。

#### 适用场景

✅ **最适合：**
- 复杂问题分析
- 需要多角度思考的问题
- 战略规划和决策
- 学术研究和深入探讨
- 代码架构设计
- 创意方案构思

❌ **不适合：**
- 简单问答
- 快速查询
- 基础操作指导
- 已有明确答案的问题

#### 使用方法

1. **开启功能**
   - 在输入框上方找到 "AI 模式" 工具栏
   - 点击 "深度思考" 按钮（带箭头图标）
   - 按钮变为蓝色表示已启用
   - 右侧会显示 "深度分析" 状态

2. **输入问题**
   ```
   ✅ 好例子：
   - "分析区块链技术的优缺点及未来发展趋势"
   - "如何设计一个高可用的微服务架构？"
   - "深入探讨人工智能的伦理问题"
   - "分析 Vue 和 React 的技术选型考虑因素"
   ```

3. **获取回答**
   - AI 会进行深入思考（可能需要更长时间）
   - 提供多角度、多层次的分析
   - 包含详细的推理过程
   - 给出全面的见解和建议

4. **查看标记**
   - 您的消息旁会显示 "深度" 紫色标签
   - 表明这条消息使用了深度思考模式

#### 功能特点

- 🎓 **深度分析**：多角度、多层次思考
- 📊 **详细推理**：展示思考过程
- 💡 **全面见解**：考虑各种可能性
- 📝 **长篇回复**：支持最多 4000 tokens

#### 与普通模式对比

| 特性 | 普通模式 | 深度思考模式 |
|------|----------|--------------|
| 回答长度 | 简洁（~2000 tokens） | 详细（~4000 tokens） |
| 思考深度 | 基础 | 深入 |
| 响应时间 | 快速 | 较慢 |
| 适用场景 | 日常问答 | 复杂分析 |
| Token 消耗 | 较少 | 较多 |

#### 技术参数

```javascript
深度思考模式配置：
- temperature: 0.8 (更有创造性)
- max_tokens: 4000 (更长回复)
- top_p: 0.95
- frequency_penalty: 0.1
```

## 🎯 组合使用

### 联网搜索 + 深度思考

将两个功能同时开启，获得最强大的 AI 能力！

#### 适用场景
- 需要最新信息的复杂分析
- 时事深度解读
- 技术趋势深入研究
- 数据驱动的战略规划

#### 示例问题
```
问题：分析 2024 年人工智能领域的发展趋势，
并预测未来 5 年的技术方向

模式：联网搜索 ✓ + 深度思考 ✓

结果：
1. AI 先搜索 2024 年最新 AI 发展动态
2. 基于搜索结果进行深度分析
3. 结合历史数据预测未来趋势
4. 提供多维度、有深度的见解
```

#### 注意事项
- ⚠️ 响应时间会更长（搜索 + 深度思考）
- ⚠️ Token 消耗更多
- ✅ 适合重要决策和深度研究

## 💡 使用技巧

### 1. 根据需求选择模式

**快速问答** → 不开启任何功能
```
问：Python 的 list 和 tuple 有什么区别？
模式：普通模式
```

**查询最新信息** → 开启联网搜索
```
问：OpenAI 最新发布了什么产品？
模式：联网搜索 ✓
```

**深度分析** → 开启深度思考
```
问：分析微服务架构的利弊
模式：深度思考 ✓
```

**综合分析** → 两者都开启
```
问：分析当前 AI 芯片市场竞争格局和未来趋势
模式：联网搜索 ✓ + 深度思考 ✓
```

### 2. 优化提问方式

#### 联网搜索模式
```
❌ 不好：什么是 AI？
✅ 好：2024 年 AI 技术有哪些突破性进展？

❌ 不好：如何学编程？
✅ 好：目前最流行的编程语言是什么？

提示：加上时间限定词（最新、当前、今年等）
```

#### 深度思考模式
```
❌ 不好：Vue 和 React 哪个好？
✅ 好：从性能、生态、学习曲线等多个维度对比 Vue 和 React，
      并给出不同场景下的选型建议

❌ 不好：什么是区块链？
✅ 好：深入分析区块链技术的原理、优势、局限性，
      以及在金融、供应链等领域的应用前景

提示：要求 AI "深入分析"、"多角度考虑"、"详细解释"
```

### 3. 功能切换

- ✅ 可以随时开启/关闭功能
- ✅ 每条消息独立记录使用的功能
- ✅ 不影响历史对话
- ✅ 可以在对话中途切换模式

### 4. 性能优化

**节省 Token：**
- 简单问题不开启深度思考
- 历史信息不需要联网搜索
- 连续对话时考虑关闭不必要的功能

**提升速度：**
- 快速查询关闭所有高级功能
- 只在需要时启用联网搜索
- 深度思考用于重要问题
