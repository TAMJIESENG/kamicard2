### 示例 1：技术教程

**提问：** "教我如何使用 async/await"

**AI 回复示例：**

## Async/Await 使用指南

### 什么是 Async/Await？

`async/await` 是 JavaScript 中处理异步操作的现代语法，它让异步代码看起来像同步代码。

### 基础用法

```javascript
// 定义异步函数
async function fetchData() {
    try {
        const response = await fetch('https://api.example.com/data')
        const data = await response.json()
        return data
    } catch (error) {
        console.error('获取数据失败:', error)
    }
}

// 调用异步函数
fetchData().then(data => console.log(data))
```

### 核心要点

1. **async 关键字**：定义异步函数
2. **await 关键字**：等待 Promise 完成
3. **错误处理**：使用 try-catch 捕获错误

> 注意：await 只能在 async 函数内使用

### 对比 Promise

| 特性 | Promise | Async/Await |
|------|---------|-------------|
| 语法 | 链式调用 | 同步风格 |
| 可读性 | 一般 | 优秀 |
| 错误处理 | .catch() | try-catch |

### 示例 2：问题排查

**提问：** "为什么我的代码报错：Cannot read property of undefined"

**AI 回复示例：**

## 错误分析

这个错误通常发生在尝试访问 `undefined` 或 `null` 对象的属性时。

### 常见原因

1. **对象未初始化**
2. **API 数据未返回**
3. **拼写错误**

### 解决方法

#### 方法 1：可选链操作符

```javascript
// 使用可选链避免错误
const value = obj?.property?.subProperty
```

#### 方法 2：默认值

```javascript
// 提供默认值
const value = obj.property || '默认值'
```

#### 方法 3：条件检查

```javascript
// 先检查再访问
if (obj && obj.property) {
    const value = obj.property
}
```

> **建议**：在访问深层属性前，始终检查对象是否存在

### 示例 3：代码对比

**提问：** "对比 Vue 2 和 Vue 3 的组件写法"

**AI 回复示例：**

## Vue 2 vs Vue 3 组件对比

### Vue 2 - Options API

```javascript
export default {
  data() {
    return {
      count: 0
    }
  },
  methods: {
    increment() {
      this.count++
    }
  },
  mounted() {
    console.log('组件已挂载')
  }
}
```

### Vue 3 - Composition API

```javascript
import { ref, onMounted } from 'vue'

export default {
  setup() {
    const count = ref(0)
    
    const increment = () => {
      count.value++
    }
    
    onMounted(() => {
      console.log('组件已挂载')
    })
    
    return { count, increment }
  }
}
```

### 主要区别

| 特性 | Vue 2 | Vue 3 |
|------|-------|-------|
| API 风格 | Options API | Composition API |
| 代码组织 | 按选项分组 | 按功能分组 |
| TypeScript | 支持一般 | 完美支持 |
| 性能 | 良好 | 更优 |
| 代码复用 | Mixin | Composables |
